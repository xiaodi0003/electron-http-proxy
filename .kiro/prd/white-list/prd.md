# 代理白名单配置页面

## UI

- src-vue 里面加一个新的页面，名称是白名单
- 参考 src-vue/views/DynamicProxy 的页面设计
- 可以增加行，每行有2列
  - 第1列：白名单的域名，是一个输入框，支持精确的域名或者通配符
  - 第2列：操作列，可以编辑、删除、启用、禁用
- 页面底部显示当前系统已有的白名单

## 行为

- 当白名单发生变更后，同步更新网络设置，让白名单的域名不要被代理
- 实现逻辑参考 server/systemShell.js
- 白名单需要持久化到本地，下次进入能看到数据，重启服务也能直接用它来设置网络
- 程序启动后，需要每5秒获取一次当前系统的白名单用于页面底部的显示
- 启动时，读取系统白名单，以系统白名单为准，如果应用设置的白名单不在系统中，则更改状态为禁用
